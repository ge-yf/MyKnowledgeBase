## 列表

Markdown 支持有序列表和无序列表。

无序列表使用星号 (\*)、加号(+)或是减号(-)作为列表标记，这些标记后面要添加一个空格，然后再填写内容。

有序列表使用数字并加上`.` 号来表示。

列表嵌套只需在子列表中的选项前面添加四个空格即可。

## 链接

常规链接(注意要使用英文符号)：  
```text
     [知乎](http://www.zhihu.com)
```

打开H盘的mao文件夹(注意要使用英文符号)：  
```text
     [猫](file:///H:/mao)
```

打开H盘的mao文件夹里的va.mp4(注意要使用英文符号)：  
```text
     [猫](file:///H:/mao/va.mp4)
```
